# Calculadora Simples

Este é um projeto de calculadora simples desenvolvido em HTML, CSS e JavaScript.

## Funcionalidades

- Adiciona números ao visor quando os botões numéricos são clicados.
- Executa operações matemáticas básicas: adição, subtração, multiplicação e divisão.
- Limpa o visor quando o botão 'C' é clicado.
- Calcula o resultado da expressão matemática quando o botão '=' ou a tecla 'Enter' é pressionado.

## Instruções de Uso

1. Clone o repositório para o seu ambiente local.
2. Abra o arquivo `index.html` em um navegador da web.
3. Utilize os botões na interface ou da calculadora ou seu teclado para inserir números e realizar cálculos.
4. Pressione a tecla 'Enter' para calcular o resultado.
5. Pressione a tecla 'C' para limpar o visor.
